(Virtualization of internet---->more specifically how group of clients attached to different router communicate with each other.) 
In this we implement DNS Simple routing algorithm (routing table, communication header)

It is a virtualization of how group of router ,which have different number of clients attached to them share their client info(with the help of routing algorithm) to make
communication between client possible.In this DNS is also implemeted(which gives virtualization of how msg packet from client sent to DNS and come back through series of routers). 
How headers are used to guide communication path.

To run it first execute servers in sequence 1,2,3,4. and then execute client
